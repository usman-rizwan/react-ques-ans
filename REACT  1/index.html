  • What is React???

    React is a free front-end JavaScript library for building user interfaces or UI components. It is maintained by Facebook and a community of individual developers. It is used as in the development of single-page ..

  • What is Single Page Application (SPA)

    A single-page application (SPA) is a web application or website that interacts with the user by dynamically , instead of the default method of a web browser loading entire new pages.

  • What is Multi Page Application (MPA)

    A web application that reloads the entire page whenever the user interacts with the website.Each time that the data is exchanged, the application makes a request from the server to get data again and again..

  • SPA VS MPA

    SPA are becoming popular day by day.. Facebook, YouTube, Twitter, GitHub, and numerous Google's services are all built using the SPA tech .. MPA are not popular yet because these websites transfer a lot of data from server to client and client to server, hence they are low on speed..

  • ANGULAR VS REACT

    AngularJS is a structural framework for developing dynamic web apps, whereas React is a javascript library that allows you to build UI components.Angular JS is based on MVC (Model View Controller) whereas React is based on Virtual DOM..

  • Library VS FRAMEWORK

    Library is a resuable functions used by computer programs. They are important in program linking and binding process...
    A Framework is a structure around which something can be built..

  • DOM VS VIRTUAL DOM

    A DOM represents the page so that programs can change the document structure, style, and content. ...
    The Virtual DOM is a light-weight abstraction of the DOM. It as a copy of the DOM, that can be updated without affecting the actual DOM..
